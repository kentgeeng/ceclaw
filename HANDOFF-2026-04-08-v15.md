## LLM Wiki POC 說明（~/llm-wiki-poc/）

### 架構
```
~/llm-wiki-poc/
├── raw/          # 原始文件（唯讀）
├── wiki/         # LLM編譯的markdown
│   ├── law/      # 法規頁
│   ├── industry/ # 產業分析頁
│   ├── culture/  # 文化習俗頁
│   ├── synthesis/# 綜合分析頁
│   ├── index.md  # 目錄
│   └── log.md    # 操作記錄
├── schema.md     # LLM讀的規則（含⚠️待驗證規則）
├── schema.yaml   # 程式讀的觸發條件
├── ingest.py     # 文件編譯
├── query.py      # Wiki查詢
└── lint.py       # 健康檢查（含無來源推論標記）
```

### 使用方式
```bash
cd ~/llm-wiki-poc

# Ingest新文件
python3 ingest.py raw/你的文件.md

# 查詢
python3 query.py "加班費怎麼算？"

# Lint健康檢查
python3 lint.py
```

## OpenShell Sandbox 說明

### 現況
```bash
# Gateway
openshell gateway start --name ceclaw-test --port 18234
# 已設為開機自動恢復（K3s in Docker）

# 查看狀態
openshell term --gateway ceclaw-test

# 連入sandbox
openshell sandbox connect ceclaw-local --gateway ceclaw-test
```

### 自訂 Dockerfile 最低要求
```dockerfile
FROM ubuntu:24.04
RUN apt-get update && apt-get install -y python3 iproute2 && rm -rf /var/lib/apt/lists/*
RUN groupadd -r sandbox && useradd -r -g sandbox -d /sandbox -s /bin/bash sandbox
RUN mkdir -p /sandbox && chown sandbox:sandbox /sandbox
USER sandbox
WORKDIR /sandbox
```

### 重要發現
- openshell-sandbox binary 由 gateway 自動 side-load，不需在 image 裡
- PersistentVolume 確保資料在 reboot 後存活（已驗證二次）
- bug #768（重開機後壞掉）在 Linux native 不存在
